# Network Visibility Overview

When you log into [Security Onion Console](security-onion-console.md), you may see alerts from [Suricata](suricata.md) or [IDH](idh.md), protocol metadata logs from [Zeek](zeek.md) or [Suricata](suricata.md), file analysis logs from [Strelka](strelka.md), or full packet capture from [Suricata](suricata.md). How is that data generated and stored? This section covers the various processes that Security Onion uses to analyze and log network traffic.
